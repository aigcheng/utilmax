    utilmax

    通用js方法库

    Installation:

    $ npm i --save utilmax

    Usage

    import * as utilmax from "utilmax";
const env = utilmax.getEnvironment();

    通常情况并不需要全量引入，所以仅添加需要的方法即可，如下：

    import { getEnvironment } from "utilmax";
const env = getEnvironment();
